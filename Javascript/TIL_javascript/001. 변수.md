## <변수 : let , var차이>
1) 공통점 :  둘 다 변수 할당 가능, 변경 가능
2) 차이점 : var는 중복선언 가능, let은 중복 선언 불가
3) 예시
var age = 25;
// 1000줄 코드
var age = 30;
-------------------------
let age =30;
let age -20; 불가

## <상수 : const>
const age = 25;
age =30;
1) 상수는 중간에 값을 못 바꿈
2) # [const로 할당하면 "age" is read-only]
3) 상수는 선언 이후에 값을 바꿀 수 없다
ex) const age; 하면 값 나중에 못 넣음
