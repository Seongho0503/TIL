### 기존의 배열 순회 방법 : for 반복문
```js
const arr = [1, 2, 3, 4];

for(let i =0; i <arr.length; i++){
  console.log(arr[i])
}
```

## 📌 01. 개념 설명
```js
const arr = [1, 2, 3, 4];

arr.forEach(function (elm) {
  console.log(elm);
});

arr.forEach((elm) => console.log(elm))
```
